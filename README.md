# 会议室调度系统 Meeting Room Scheduling System

## 项目简介
一款面向企业/校园场景的会议室预约与管理平台，解决会议室资源分配混乱、预约冲突频发的问题，提供一站式会议室资源调度方案。

## 核心功能
- 会议室信息管理（状态维护、设备配置）
- 在线预约申请与自动冲突检测
- 预约审批流与权限分级管理
- 使用数据统计与可视化看板

## 技术栈
- 前端：Vue.js / Element UI / ECharts
- 后端：Python / Flask
- 数据库：MySQL
